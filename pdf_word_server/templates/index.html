PDF to DOCX over LAN

Chuyển PDF sang Word ngay trên máy nội bộ

Server này chạy cục bộ trong mạng LAN và convert PDF sang .docx ngay trên máy chủ. Cấu hình hiện tại dùng engine {{ settings.converter_engine }} để ưu tiên giữ bố cục gần với file gốc nhất trong phạm vi convert tự động.

Giới hạn upload {{ settings.upload_limit_mb }} MB
Cơ chế convert {{ settings.converter_engine }}
Mạng truy cập LAN / nội bộ

Sẵn sàng

Tải file PDF lên để tạo file Word và tải về ngay trên trình duyệt.

Phù hợp nhất với PDF dạng text hoặc PDF xuất từ Word/Excel/PowerPoint.

PDF scan ảnh vẫn có thể convert, nhưng mức độ giữ format và khả năng sửa text sẽ phụ thuộc chất lượng file nguồn.

Server xử lý tuần tự từng file để tránh lỗi khi nhiều client upload cùng lúc.
